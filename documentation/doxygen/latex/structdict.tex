\doxysubsection{Detailed Description}
Element of a dictonary as double linked list. 

\begin{DoxyVerb}-next and prev are the pointers to maintain the linked list
-type stores the data-type, stored in value
-value stores the data
-key stores the dictonary key, which is always a string in JSON.
\end{DoxyVerb}
